This project focuses on **Server-Side Rendering (SSR)** using **Python** and the **Flask** framework.  
Server-side rendering generates complete HTML pages on the server and sends them to the client, improving performance, SEO, and maintainability compared to client-side rendering.

The project uses the **Jinja2 templating engine** to dynamically generate HTML and integrates multiple data sources such as **JSON**, **CSV**, and **SQLite databases**.
