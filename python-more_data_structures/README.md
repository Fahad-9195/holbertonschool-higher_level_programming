# Python - More Data Structures: Set, Dictionary

This project continues the exploration of Python data structures with a focus on **sets**, **dictionaries**, and advanced operations such as **lambda**, **map**, **filter**, and dictionary comprehension.

## Learning Objectives

By the end of this project, you should be able to explain, without Google:

### General
- What sets are and how to use them.
- The most common set methods and how to use them.
- When to use sets versus lists.
- How to iterate through a set.
- What dictionaries are and how to use them.
- What keys, values, and items are.
- How to loop through a dictionary.
- What lambda functions are and how to use them.
- How to use `map()`, `filter()`, and `reduce()`.
- How to use dictionary comprehensions.
- How to handle nested data structures.
