# Python - Data Structures: Lists, Tuples

This project is part of the Holberton School curriculum for higher-level programming.  
It focuses on understanding and using Python data structures, specifically **lists**, **tuples**, and **sequences**.

## Learning Objectives

By the end of this project, you should be able to explain:

### General
- What lists are and how to use them.
- The differences and similarities between lists and strings.
- The most common list methods and how they are used.
- How to use lists as stacks and queues.
- What list comprehensions are and how to use them.
- What tuples are and how to use them.
- When to use tuples versus lists.
- What sequences are.
- What tuple packing is.
- What sequence unpacking is.
- What the `del` statement is and how to use it.
