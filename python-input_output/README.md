# Python - Input/Output

## Description
This project covers file handling in Python, including reading and writing files, using context managers, and working with JSON serialization and deserialization.

## Learning Objectives
By the end of this project, you should be able to explain:

- How to open, read, and write files in Python
- How to read a file line by line
- How to move the cursor inside a file
- How to properly close a file
- How to use the `with` statement
- What JSON is
- What serialization and deserialization mean
- How to convert Python objects to JSON
- How to convert JSON data to Python objects
- How to use command line arguments in Python
